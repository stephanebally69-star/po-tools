# PO Tools — Claude Code Context

## Projet
Boîte à outils pour Product Owner. Contient des templates d'US, des plans de recette, des scripts d'automatisation Azure DevOps et des skills Claude Code.

## Structure
- `templates/user-stories/` — Templates de User Stories (standard et Gherkin)
- `templates/recette/` — Plans de recette et checklists
- `scripts/azure-devops/` — Scripts d'interaction avec l'API Azure DevOps
- `skills/` — Skills Claude Code pour automatiser les tâches PO

## Conventions
- Les templates sont en markdown
- Les scénarios Gherkin sont en français (Etant donné / Quand / Alors)
- Les scripts Azure DevOps utilisent l'API REST ou le CLI `az boards`
