# Skills Claude Code

Skills réutilisables pour les tâches PO récurrentes.

## Skills disponibles

| Skill | Description |
|-------|------------|
| `generer-us` | Génère une User Story complète à partir d'un brief |
| `generer-recette` | Génère un plan de recette à partir d'une US |
| `review-us` | Revoit et améliore une US existante |

## Utilisation

Les skills sont invocables dans Claude Code pour accélérer le travail quotidien du PO.
